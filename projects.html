Pient

Pient is an ease-of-use pip tool for python development.
Pient is capable of:

- automatically updating your pip packages
- list all downloaded packages
- update pip
- and obviously, download and uninstall pip packages.

i mainly made this to learn config files. feel free to leave some suggestions!
